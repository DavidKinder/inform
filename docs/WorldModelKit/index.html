  • WorldModelKit

Inter-level support for modelling space, time and actions in interactive fiction. This is version 1.

  • Miscellany - Miscellany

  • State - State To implement out of world actions relating to the program state.

  • Light - Light The determination of light, visibility and physical access.

  • ListWriter - ListWriter A flexible object-lister taking care of plurals, inventory information, various formats and so on.

  • OrderOfPlay - OrderOfPlay The sequence of events in play: the Main routine which runs the startup rulebook, the turn sequence rulebook and the shutdown rulebook; and most of the I6 definitions of primitive rules in those rulebooks.

  • Actions - Actions To try actions by people in the model world, processing the necessary rulebooks.

  • Figures - Figures To display figures and play sound effects.

  • OutOfWorld - OutOfWorld To implement some of the out of world actions.

  • Printing - Printing To manage the line skips which space paragraphs out, and to handle the printing of pieces of text and numbers.

  • WorldModel - WorldModel Testing and changing the fundamental spatial relations.

  • MapRouteFinding - Testing and changing the fundamental spatial relations.

  • Time - Time Support for parsing and printing times of day.

  • Tests - Tests The command grammar and I6 implementation for testing commands such as TEST, ACTIONS and PURLOIN.

  • Chronology - Chronology To record information now which will be needed later, when a condition phrased in the perfect tense is tested.

  • StoredAction - StoredAction Code to support the stored action kind of value.
